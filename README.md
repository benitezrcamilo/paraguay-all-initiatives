# PY Paraguay — Official GitHub Repositories 🌎

MapBiomas Paraguay is an initiative that monitors land use and land cover changes across the national territory through annual multitemporal mapping. It helps identify pressures on ecosystems and the expansion of human-modified areas. To generate annual maps, the project relies on a network of experts and tools such as Google Earth Engine. The maps cover the entire Paraguayan territory and include specialized themes focused on key environmental monitoring products such as fire and water dynamics across the country.

This repository aggregates other **MapBiomas Paraguay** repositories dedicated to thematic collections, cross-cutting themes, mosaics, validation workflows, and analytical modules.

All repositories listed below are maintained by the [**MapBiomas Paraguay**](https://paraguay.mapbiomas.org/) team and follow the official **MapBiomas methodological framework**. Below you will find links to the main code repositories used in our workflows.

## 🚀 Getting Started

To run most of the scripts you will need:

- A [Google Earth Engine](https://earthengine.google.com/) account
- Access to the required assets (some are public, others must be copied to your account)
- Follow the instructions in each repository's `README.md` to set output paths and required modules
- Some scripts might be in Python hence you'll need access to Earth Engine API in order to run it but the main steps works in Google Earth Engine interface using JavaScript.

If you are not familiar with MapBiomas workflows, check the [official methodology documentation](https://paraguay.mapbiomas.org/descripcion-general-de-la-metodologia/).
